# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2025-12-23

### Added

- Initial release of **AZtomiq Core**.
- Atomic tool architecture.
- Multi-language support (English & Vietnamese).
- PWA support.
- Dark mode.

### Changed

- Rebranded from ZTools to **AZtomiq**.
- Updated UI/UX with premium aesthetic.
- Switched to Lucide icons.
- Refactored local storage keys for better naming consistency.
